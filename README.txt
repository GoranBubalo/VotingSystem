# Voting Smart Contract Workspace

This workspace contains:

- `contracts/Voting.sol` — a smart contract for secure voting.
- *(Optional)* `scripts/` — deploy scripts for Voting (if added).
- *(Optional)* `tests/` — tests for Voting (if added).

## How to use

1. Compile `Voting.sol`  
2. Deploy using Remix Deploy tab or your own deploy script.  
3. Interact with functions: start voting, end voting, vote, get winner.
